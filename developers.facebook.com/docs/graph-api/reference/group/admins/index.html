Graph API Version
v2.12

/{group-id}/admins

The admins of a group. This edge is only available to app and game groups.

Reading

An Admin of the Group can get a list of other Group Admins.

Permissions

A User access token of an Admin of the Group.

Permissions

  • An app access token can make any existing group member an admin when the group was created by that app.

Fields

Name Description Type

member

The user to make an admin. They must be a member of the group and user of the app. This is required.

string

Response

If successful:

{
  "success": true
}

Otherwise a relevant error message will be returned.
