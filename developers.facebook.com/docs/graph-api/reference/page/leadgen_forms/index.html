Page Leadgen Forms

Reading

A library of lead generation forms created for this page.

Permissions

This operation requires a Page access token.

Parameters

This endpoint doesn't have any parameters.

Fields

Reading from this edge will return a JSON formatted result:

{
    "data": [],
    "paging": {}
}

data

A list of LeadGenData nodes.

paging

For more details about pagination, see the Graph API guide.

Validation Rules

ErrorDescription
100 Invalid parameter
200 Permissions error

Creating

You can make a POST request to leadgen_forms edge from the following paths:
  • /{page_id}/leadgen_forms
When posting to this edge, no Graph object will be created.
