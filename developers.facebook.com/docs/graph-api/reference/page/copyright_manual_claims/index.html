Page Copyright Manual Claims

Reading

You can't perform this operation on this endpoint.

Creating

You can make a POST request to copyright_manual_claims edge from the following paths:
  • /{page_id}/copyright_manual_claims
When posting to this edge, a VideoCopyrightMatch will be created.
