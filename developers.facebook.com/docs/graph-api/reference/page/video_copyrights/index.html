Fields

Reading from this edge will return a JSON formatted result:

{
    "data": [],
    "paging": {},
    "summary": {}
}

data

A list of VideoCopyright nodes.

paging

For more details about pagination, see the Graph API guide.

summary

Aggregated information about the edge, such as counts. Specify the fields to fetch in the summary param (like summary=total_count).

FieldDescription

total_count

unsigned int32

Total number of reference files on the page

Validation Rules

ErrorDescription
100 Invalid parameter

Creating

You can make a POST request to video_copyrights edge from the following paths:
  • /{page_id}/video_copyrights
When posting to this edge, a VideoCopyright will be created.

Permissions
