Page Call To Action

A page access token with pages_manage_cta permission is required for creating, updating and deleting.

Reading

Page's call-to-action

Creating

You can make a POST request to call_to_actions edge from the following paths:
  • /{page_id}/call_to_actions
When posting to this edge, a PageCallToAction will be created.
