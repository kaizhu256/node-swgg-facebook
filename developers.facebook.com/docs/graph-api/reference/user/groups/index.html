User Groups

A list of Facebook Groups of which a person is an admin.

Permissions

You need to ask for the user_managed_groups permission to retrieve any groups the session user admins.

Login Review: user_managed_groups

To use the user_managed_groups permission you need to submit your app for review.

Learn more about Login Review

Reading

The Facebook Groups of which the person is an Admin.

Parameters

NameDescription
parent
numeric string or integer

Filter to see all children groups of a particular community group, by setting to the community group's ID

Fields

Reading from this edge will return a JSON formatted result:

{
    "data": [],
    "paging": {}
}

data

A list of Group nodes.

The following fields will be added to each node that is returned:

FieldDescription

administrator

bool

Shown if this person is an admin of the group.

bookmark_order

unsigned int32

This group's place in the list of bookmarks on this person's Facebook homepage.

unread

unsigned int32

A count of the number of unread posts in that group.

paging

For more details about pagination, see the Graph API guide.
