Permissions

Developers usually request these permissions for this endpoint:

Marketing Apps
No data
Page management Apps
No data
Other Apps
Permissions are not usually requested.

Parameters

This endpoint doesn't have any parameters.

Fields

Reading from this edge will return a JSON formatted result:

{
    "data": [],
    "paging": {}
}

data

A list of FavoriteRequest nodes.

paging

For more details about pagination, see the Graph API guide.

Validation Rules

ErrorDescription
200 Permissions error
110 Invalid user id

Creating

You can make a POST request to favorite_requests edge from the following paths:
  • /{user_id}/favorite_requests
When posting to this edge, a FavoriteRequest will be created.
