Graph API Reference /{user-id}/home

As of October 6th, 2015, this endpoint is no longer available. Please consider using the /user-id/feed edge instead.

Please see our Changelog for more information.

This endpoint returns posts a person sees in their News Feed. The posts returned by this API may not be identical to the posts a person would see on facebook.com or in Facebook's mobile apps.

Permissions

  • A user access token with read_stream permission is required to view that person's news feed.

Modifiers

NameDescriptionType

with

Use location to retrieve only posts with a location attached.

enum{location}

filter

Retrieve only posts that match a particular stream filter. Valid filters to be used here can be retrieved using the FQL stream_filter table.

string

Fields

An array of Post objects.

Publishing

Use the /{user-id}/feed edge to create new posts.

Deleting

You can't delete using this edge.

Updating

You can't update using this edge.
