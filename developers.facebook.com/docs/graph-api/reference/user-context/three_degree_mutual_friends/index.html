User Context: Three Degree Mutual Friends

Friends of the viewer who are mutual friends with the target and have the app installed.

Access to the Three Degree Mutual Friends API is restricted to a limited set of partners and usage requires prior approval by Facebook. You cannot apply to use the API at this time.

Reading
