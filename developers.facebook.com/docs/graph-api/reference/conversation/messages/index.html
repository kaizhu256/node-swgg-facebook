The messages in a conversation between a person and a page.

Permissions

  • A page access token with read_page_mailboxes permission is required to view any messages in a conversation that Page is involved in.
  • A user access token with read_mailbox permission can be used to view any messages in a conversation that person is involved in. The person must be a developer of the app making the request with this token.

Permissions

Pages can only reply to a message. They cannot initiate a conversation.

  • A page access token for the page is required to publish a message.

Fields

Parameter Description Type

message

The text of reply. This field is required.

string

Response

Name Description Type

id

The ID of the newly created message.

string

A page admin can privately reply to a user who has commented on a post on the page however a system message will be returned by the Graph API.
