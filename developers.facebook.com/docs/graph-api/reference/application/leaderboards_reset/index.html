Application Leaderboards Reset

Reading

You can't perform this operation on this endpoint.

Creating

You can make a POST request to leaderboards_reset edge from the following paths:
  • /{application_id}/leaderboards_reset
When posting to this edge, no Graph object will be created.

Parameters

NameDescription
name
string

The name of the leaderboard.

Required
reset_time
datetime/timestamp

Timestamp that the leaderboard should be reset to. All scores set before the reset time will be invalidated.

Required
