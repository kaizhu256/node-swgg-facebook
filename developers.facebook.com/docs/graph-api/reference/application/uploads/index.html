Application Uploads

Please see our guide on the Resumable Upload API for detailed information about using this API.

Reading

You can't perform this operation on this endpoint.

Creating

Create an upload session. The session will be used to upload a file via resumable upload endpoint. The retrieved handle can be specified in place of an uploaded file among other Graph API calls.

You can make a POST request to uploads edge from the following paths:
  • /{application_id}/uploads
When posting to this edge, no Graph object will be created.

Parameters

NameDescription
file_length
int64

The file length in bytes
file_name
string

The name of the file to be uploaded
file_type
string

The MIME type of the file to be uploaded
session_type
enum {attachment}
Default value: attachment

The type of upload session that is being requested by the app
